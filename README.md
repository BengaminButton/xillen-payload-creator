# Xillen Payload Creator

Инструмент для создания полезных нагрузок (payloads) для тестирования безопасности.

## Возможности
- Генерация различных payload
- Поддержка кастомизации

## Быстрый старт
```bash
python main.py
```

## Требования
- Python 3.x
